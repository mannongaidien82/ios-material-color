■ Description
This is Material Color layout color values.

■ Setup
 Prefix.pch into your project folder.
 And use it.

 backgroundColor = MC_RED_50;

■ Example
UIButton *tempButton = [[UIButton alloc] init];
tempButton.backgroundColor = MC_RED_50;
